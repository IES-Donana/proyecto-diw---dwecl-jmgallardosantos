mis proyectos

Con este proyecto lo que se pretende es poner en marcha un observatorio astronómico controlado remotamente desde cualquier dispositivo (Pc, Tablet, móvil) y sistema operativo (Linux, Microsoft, Ios). Compuesto por un telescopio computerizado, varias cámaras, cúpula, estación meteorológica y un servidor que muestre una web a los visitantes con imagen y datos diarios del observatorio así como una base .

He creado una aplicacion que permita a los pacientes del centro Oncológico Donostia conectarse a sus bases de datos para poder hacer unas determinadas consultas. La aplicacion esta creada de tal forma que puede ser fácilmente ampliada para que los profesionales del centro y los profesionales de las mutuas.

El proyecto se basa en la creación de un videojuego mediante el motor Unity, tratando de que sea lo más claro, sencillo y atractivo posible para el usuario, y que además posea una alta rejugabilidad, lo cual se realiza mediante un sistema de aleatoriedad de niveles y objetivos a lo largo de la partida. El proyecto es multiplataforma, pudiéndose jugar tanto en PC como en sistemas Android.

Gestareas es un proyecto que cubre varias necesidades de las empresas que ofrecen servicios denominadas «de campo» o también «facility services». La aplicación consta de 4 mini proyectos en 1, que serian: un servidor de Microsoft Azure (Linux Centos) con toda su implementación y configuración (web, base de datos, etc), un panel de control vía web, una app móvil multiplataforma desarrollada con Javascript y un api o web service.
